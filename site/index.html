omnilimb-face

A face & voice for your hermes agent — a Live2D / Live3D VTuber plugin with hands-free voice, real-time barge-in, lip-sync and expressions.给你的 hermes 智能体一张会说话的脸 —— 带语音免提、实时打断、口型同步与表情驱动的 Live2D / Live3D 形象插件。

What it does它能做什么

Hands-free voice语音免提

Talk to your agent with VAD + STT — no push-to-talk needed.VAD + STT 自动断句,免按键直接说话。

Real-time barge-in实时打断

Speak while the avatar talks to interrupt it instantly.形象说话时你一开口即可立刻打断。

Live2D & Live3D avatarLive2D 与 Live3D 形象

Switch at runtime between a 2D Cubism model and a 3D VRM model (three-vrm) — both driven by the reply audio with lip-sync and expressions.可在运行时切换 2D Cubism 模型与 3D VRM 模型(three-vrm)—— 两者都由回复音频驱动口型与表情。

Uses your real agent复用你的真实智能体

Reuses your hermes model, tools and memory — it never calls an LLM itself.复用你的 hermes 模型/工具/记忆,自己从不调用 LLM。

No core changes不改核心

Integrates only via register(ctx); degrades gracefully if extras are missing.仅经 register(ctx) 集成;缺少可选依赖时优雅降级。

Mobile ready支持手机端

Runs on a phone over HTTPS (self-signed cert) so the mobile mic works on the LAN.可经 HTTPS(自签证书)在手机上运行,局域网内手机麦克风可用。

Quick start快速开始

A standalone, installable plugin for hermes-agent. Requires Python 3.11+. Two ways to run it:

面向 hermes-agent 的独立可安装插件。需要 Python 3.11+。两种运行方式:

A. 1-minute preview (no hermes-agent needed):

A. 一分钟预览(无需 hermes-agent):

# 1) install EVERYTHING in one command (incl. voice, so you hear it)
pip install -e ".[all]"
# 2) start it (serves the web page + gateway)
python preview.py
# 3) open the PAGE:  http://127.0.0.1:12394/     (NOT 12393)

B. Full product — chat in hermes, the avatar speaks the real replies:

B. 完整形态 —— 在 hermes 里聊天,形象说出真实回复:

# 1) install into the hermes venv, everything in one command
<hermes-venv>/python -m pip install -e "path/to/omnilimb-face[all]"
# 2) enable it: add `omnilimb-face` to plugins.enabled in ~/.hermes/config.yaml
# 3) run hermes (starts gateway 12393 + front-end 12394)
hermes
hermes vtuber status
# 4) open  http://127.0.0.1:12394/  and chat in the terminal

⚠️ The web page is on port 12394; 12393 is the WebSocket gateway (opening 12393 in a browser won't work). No sound? install the voice engine via ".[all]" (or pip install edge-tts) and click the page once. Full docs, config, CLI / slash commands, Live2D↔Live3D switching, mobile HTTPS and troubleshooting on GitHub.⚠️ 网页在 12394 端口;12393 是 WebSocket 网关(用浏览器开 12393 打不开)。没声音?用 ".[all]"(或 pip install edge-tts)装上语音引擎,并先点一下页面。完整文档、配置、CLI/斜杠命令、Live2D↔Live3D 切换、手机 HTTPS 与故障排查见 GitHub。

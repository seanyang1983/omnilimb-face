omnilimb-face

A face & voice for your hermes agent — a Live2D / Live3D VTuber plugin with hands-free voice, real-time barge-in, lip-sync and expressions.给你的 hermes 智能体一张会说话的脸 —— 带语音免提、实时打断、口型同步与表情驱动的 Live2D / Live3D 形象插件。

What it does它能做什么

Hands-free voice语音免提

Talk to your agent with VAD + STT — no push-to-talk needed.VAD + STT 自动断句,免按键直接说话。

Real-time barge-in实时打断

Speak while the avatar talks to interrupt it instantly.形象说话时你一开口即可立刻打断。

Live2D / Live3D avatarLive2D / Live3D 形象

Lip-sync and expressions driven by the reply audio.由回复音频驱动口型与表情。

Uses your real agent复用你的真实智能体

Reuses your hermes model, tools and memory — it never calls an LLM itself.复用你的 hermes 模型/工具/记忆,自己从不调用 LLM。

No core changes不改核心

Integrates only via register(ctx); degrades gracefully if extras are missing.仅经 register(ctx) 集成;缺少可选依赖时优雅降级。

Mobile ready支持手机端

Runs on a phone over HTTPS (self-signed cert) so the mobile mic works on the LAN.可经 HTTPS(自签证书)在手机上运行,局域网内手机麦克风可用。

Quick start快速开始

A standalone, installable plugin for hermes-agent. Requires Python 3.11+.

面向 hermes-agent 的独立可安装插件。需要 Python 3.11+。

# install into the hermes venv (editable, no dependency changes)
<hermes-venv>/python -m pip install -e path/to/omnilimb-face --no-deps
<hermes-venv>/python -m pip install edge-tts          # keyless TTS fallback
# enable it: add `omnilimb-face` to plugins.enabled in ~/.hermes/config.yaml
hermes                       # starts the /client-ws gateway + front-end
hermes vtuber status

Full docs, configuration, CLI / slash commands, Live2D↔Live3D switching, mobile HTTPS setup and troubleshooting are on GitHub.完整文档、配置、CLI/斜杠命令、Live2D↔Live3D 切换、手机 HTTPS 配置与故障排查见 GitHub。
